***** Woocommerce bulk discount Helper with CSV  importing *******

How to Use:
1. Make sure modified woocomerce bulk discount plugin is installed. It is included with the package
2. Upload the plugin and activate it
3. Create a CSV using SKU and message, Please take a look at the sample csv included with the package
4. Browe the upload page by finidng the "Bulk Discount" submneu  under "Products" menu
5. Make sure the woocommerce bulk discount will not get updated in future. If somehow it is updated, you will 
	get a working version inside this plugin. Browse Mother Plugin and copy the woocommerce bulk uploader to 
	the plugins directory. That's all
